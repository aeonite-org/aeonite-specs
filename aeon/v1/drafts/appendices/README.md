# Appendices

Appendices are supporting documents for the consolidated AEON v1 spec set.

Authority rule:

- Canonical rules live in the top-level v1 specs and companion references.
- Appendices are informative unless a canonical document explicitly delegates a topic to an appendix.
- If an appendix conflicts with a canonical v1 document, the canonical v1 document wins.
