# Appendix — Third-Party Validator Conformance

Status: pointer to CTS-owned document

The operational conformance guidance for third-party validators now lives with `aeonite-org/aeonite-cts`.

This appendix remains in the spec tree only to record the boundary:

- AEOS and AEON specs define the normative semantic requirements
- `aeonite-cts` owns the public validator conformance guidance and verification-facing contract surface

If validator guidance conflicts with the AEOS spec line, the AEOS spec line wins.
